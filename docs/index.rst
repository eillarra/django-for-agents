===========================
Welcome to django-for-agents
===========================

django-for-agents adds agent-friendly delivery features to Django projects:

- Markdown responses via ``Accept: text/markdown``.
- Optional debug markdown toggle in development.
- Automatic ``llms.txt`` endpoint discovery/index.
- Frontmatter metadata and markdown output normalization.
